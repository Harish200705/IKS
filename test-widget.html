How to Use:

Click the circular chat icon in the bottom-right corner to open the AI Health Assistant. You can ask questions about animal diseases and treatments.

Features Implemented:

  • Circular chatbot icon fixed to bottom-right corner
  • Floating chat window with smooth animations
  • Header: "AI Health Assistant"
  • Scrollable conversation area
  • Text input with Send button
  • Modern CSS with rounded corners and shadows
  • Responsive design for mobile devices
  • In-memory caching for instant responses
  • localStorage persistence across page reloads
  • Message normalization (whitespace/punctuation insensitive)
  • Duplicate request prevention
  • API integration with specified endpoint
  • Error handling with retry functionality
  • Typing indicator while waiting for responses
  • Visual "cached" badge for cached responses
  • Auto-scroll to newest messages
  • Cache size limit (200 entries)
Note: If you encounter CORS errors, the server needs to have flask_cors.CORS(app) enabled to allow cross-origin requests.

Sample Questions to Try:

  • "What are the symptoms of fever in animals?"
  • "How to treat diarrhea in calves?"
  • "What causes watery eyes in animals?"
  • "Prevention methods for animal diseases"
